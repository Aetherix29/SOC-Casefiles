# SOC Casefiles

This repository contains a collection of Security Operations Center (SOC) incident investigations performed as part of hands-on cybersecurity training and simulations.

Each case represents a real-world inspired security alert, investigated using structured analysis methodologies similar to those used by L1 SOC Analysts.

---

## Objective

* Practice **alert triage and investigation**
* Develop **log analysis and correlation skills**
* Build **incident reporting experience**
* Map findings to the **MITRE ATT&CK framework**
* Simulate real-world **SOC workflows**

---

## Methodology

Each case follows a consistent investigation process:

1. **Alert Review** – Understand the triggered alert
2. **Log Analysis** – Examine relevant system/SIEM logs
3. **Correlation** – Identify related events and behaviors
4. **Timeline Creation** – Reconstruct sequence of activity
5. **Classification** – True Positive / False Positive
6. **MITRE Mapping** – Identify attack techniques
7. **Response Recommendation** – Suggest next steps
